# Faith Life Evangelistic Church Theme ✝️🌍
A beautifully crafted dark mode Chrome theme created for and inspired by Faith Life Evangelistic Church (FLEC). 
Featuring a radiant golden cross over the world globe, this theme brings the presence of God into your everyday browsing experience.

*"For God so loved the world that He gave His one and only Son"*
*— John 3:16*

## Install from Chrome Web Store ⬇️

👉 [Click here to install](https://chromewebstore.google.com/detail/faith-life-church-theme)

> *(Update this link once published on Chrome Web Store)*

## Features
- ✝️ Radiant golden cross centrepiece
- 🌍 World globe background — reflecting FLEC's global mission
- 🌑 Deep dark mode design — easy on the eyes
- ⭐ Warm starry brown frame
- 🏅 FLEC logo proudly at the heart of the theme
- 🎨 Clean, elegant and spiritually meaningful design
- 👁️ Distraction-free browsing environment

## About FLEC
Faith Life Evangelistic Church (FLEC) is a Spirit-filled, Bible-believing church with a burning passion to spread the Gospel of Jesus Christ to the ends of the earth. 
This theme is a digital expression of that mission — taking the message of the cross into every corner of the internet, 
one browser at a time.

## Manual Installation
1. Download or clone this repository
2. Open Chrome and go to `chrome://extensions`
3. Enable **Developer Mode** (top right toggle)
4. Click **"Load unpacked"**
5. Select the downloaded folder
